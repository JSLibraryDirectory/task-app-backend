Hapi MongoDB starter app with Lab for testing
===

> This repo is for getting started building a tech stack with core tools like Node.js, Hapi v17 and MongoDB with Mongoose ORM


### Seed some tasks

``` 
$ node src/utils/seed.js
```

### Start server

``` 
$ npm run start
```
